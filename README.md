## GitHub Pages version

The static GitHub Pages-ready site is in `docs/`.

To publish it with GitHub Pages:

1. Push this repository to GitHub.
2. Open repository `Settings`.
3. Go to `Pages`.
4. Set `Source` to `Deploy from a branch`.
5. Choose branch `main` and folder `/docs`.
6. Save.

GitHub will publish `docs/index.html` as the website.

## Local preview

You can open `docs/index.html` directly in a browser.

## Original editable app

The original React/Vinext source is still in `app/`. The `docs/` folder is the
plain static version prepared for GitHub Pages.
